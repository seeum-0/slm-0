# Sentiment Analysis Web App

A web application that performs sentiment analysis using a Hugging Face transformer model. Built with Python, Flask, HTML, Tailwind CSS, and JavaScript.

## Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/slm-project.git
cd slm-project
```

### 2. One time setup

Double-click "setup.bat"

### 3. Run the project

Double-click "run.bat"

### 4. Open in browser

Go to: http://127.0.0.1:5000

---

## How It Works

Enter a word or sentence into the input to get the sentimental value which will either be a "POSITIVE" or "NEGATIVE".

## Example

**Input:** `I love this project`  
**Output:** `POSITIVE (0.999)`
